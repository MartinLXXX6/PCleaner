# How to Run
- Navigate to the Release section and download the latest "PCleaner.zip" file
- Extract the "PCleaner.zip" file
- Open the folder
- Run PCleaner.exe (Requires Admin Privileges)

# Requirements
- Windows
- x86 or x64
